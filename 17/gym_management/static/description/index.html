Gym Management Module

A comprehensive system for managing gym memberships, workout plans, diet plans, and health tracking.

Overview

This module allows trainers and members to manage accounts, memberships, workouts, diets, and attendance in an integrated system. Track health status, manage workout and diet plans, and streamline gym management operations.

User Flow

  1. Create User Account: Upon account creation, users select whether they are a Trainer or Member.
  2. Create Membership: Admins can create Membership and Personal trainer, defining durations associated with each type.
  3. Access Membership: Members can purchase a membership and select a personal trainer.
  4. View and Add Exercises: Users can browse available exercises and create personalized workout plans.
  5. Manage Workout Plans: Create and edit workout plans, adding exercises through a simple interface.
  6. Diet Plan Management: Trainers can create personalized diet plans, and members can track their meal plans.
  7. Health Monitoring: Members can track their BMI and overall health by inputting height and weight.
  8. Attendance Tracking: View attendance records for gym sessions, accessible by both trainers and members.
  9. Invoice Tracking: Track payments and view invoices for services and memberships by admin and members.

Key Features

  • Account management for trainers and members
  • Seamless membership management with trainer assignment
  • Exercise browsing and workout plan creation
  • Diet plan management and health tracking
  • Attendance tracking and reporting

Installation Instructions

  1. Download the Gym Management Module from the Odoo App Store.
  2. Place the module folder in your Odoo addons directory.
  3. Restart your Odoo server.
  4. Navigate to the Apps menu and update the app list.
  5. Search for "Gym Management Module" and click Install.

Usage Instructions

Once installed, follow these steps to start managing the gym:

  1. Create a user account and select the user type (Trainer or Member).
  2. Members can purchase memberships, assign trainers, and view membership details.
  3. Trainers can create personalized workout and diet plans for members.
  4. Members can track their BMI and health status by updating their profile with height and weight data.
  5. Trainers and members can track attendance and view gym session history.
